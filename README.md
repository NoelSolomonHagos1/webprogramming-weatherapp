# WeatherApp Project

This is a web-based weather application built as a course project for the "Introduction to Web Programming" course.  
The app allows users to:

- Search for the weather in any city worldwide
- View forecasts for the next 3 hours, daily, and weekly
- Get weather details such as temperature, wind speed, rain, and UV index
- Use current location via Geolocation API
- Save favorite locations for quick access
- Switch between Celsius, Fahrenheit, and Kelvin units

## Technologies Used

- HTML, CSS, JavaScript
- OpenWeatherMap API & Open-Meteo API
- Geolocation API
- VS Code with Live Server extension
- Modern web browsers: Chrome, Firefox, Safari, Edge

## Features

- Responsive design (works on desktop and mobile)
- Interactive UI with hover effects for forecasts
- Clear and organized project structure
- Own feature: UV index warning if high, and interactive forecast elements

## Notes

- The app is easy to navigate, no manual required
- Hourly forecast uses 3-hour intervals due to API restrictions
